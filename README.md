Simple Express File Upload API

A simple Express.js API for securely uploading image files.

📁 Directory Structure
file-upload-api/
├── uploads/              # Where valid uploaded images are saved
├── node_modules/         # Installed dependencies (appears after install)
├── index.js              # Main Express server code
├── package.json          # Project settings and dependencies
└── README.md             # This file
🚀 How to Install and Run
1. Install Node.js

Make sure you have Node.js installed on your computer.

2. Navigate to the project folder
cd file-upload-api
3. Install dependencies

This will install Express, Multer, and file-type:

npm install
4. Start the server
npm start
🌐 Server URL

The server will start and listen for uploads at:

http://localhost:3000
