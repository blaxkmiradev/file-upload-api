Simple Express File Upload API

A simple Express.js API for securely uploading image files.

Directory Structure

file-upload-api/
├── uploads/              # Where valid uploaded images are saved
├── node_modules/         # Installed dependencies (appears after install)
├── index.js              # Main Express server code
├── package.json          # Project settings and dependencies
└── README.md             # This file


How to Install and Run

Install Node.js: Make sure you have Node.js installed on your computer.

Open your terminal and navigate to your project folder:

cd file-upload-api


Install the dependencies (Express, Multer, and file-type):

npm install


Start the server:

npm start


The server will start and listen for uploads on http://localhost:3000.
